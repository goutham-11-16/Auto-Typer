# Auto Typer byGo

Auto Typer is a powerful, offline Windows application designed to boost your productivity by automating text insertion. Create custom snippets, assign hotkeys, and type complex text sequences instantly.

## Features

*   **Custom Hotkeys**: Trigger text snippets with any key combination (e.g., `F8`, `Ctrl+Alt+T`).
*   **Rich Macros**: Insert special keys like `{ENTER}`, `{TAB}` or add `{DELAY 500}` for precise control.
*   **Typing Modes**:
    *   **Human-Like**: Simulates natural typing speed.
    *   **Paste**: Instantly injects large blocks of text.
*   **Safety First**:
    *   **Global Pause**: Toggle the service on/off with `Ctrl+Alt+P`.
    *   **Cancel**: Press `ESC` strictly to stop any running typing job.
*   **Modern UI**: Clean, card-based interface with a built-in help guide.
*   **Offline Privacy**: All data is stored locally. No keylogging, no cloud upload.

## Getting Started

1.  **Download & Run**: Launch `AutoTyper.exe`.
2.  **Add Snippet**: Click the **+ Add** button.
3.  **Configure**:
    *   Enter a **Name** and the **Text** you want to type.
    *   Click **Hotkey Trigger** and press your desired key combination.
4.  **Save**: Click **Save Snippet**.
5.  **Type**: Open any application (Notepad, Browser, etc.) and press your hotkey!

## usage

*   **Macro Toolbar**: Use the chips above the editor to insert macros.
*   **Status Bar**: Check the bottom right "Service Running" indicator. clickable to pause.

## Requirements

*   Windows 10/11
*   .NET 8.0 Runtime (or included in self-contained build)
